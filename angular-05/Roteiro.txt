Roteiro
=======

1. [formGroup]="usuarioGroup" no tag form substitui  #f="ngForm"


2. (ngSubmit)="onSubmit(usuarioGroup.valid)" para submetermos a informação se o formulário está válido


3. formControlName="nome" substitui [(ngModel)]="usuarioForm.nome" e #nome="ngModel"

   formControlName="cpf"

   formControlName="dataNasc"

   formControlName="salario"


4. As mensagens de erro serão definidas na classe componente. As linhas abaixo exibem todas as mensagens de erro para
   nome.

   <div *ngIf="formErrors.nome && submitted" class="alert alert-danger">
       {{ formErrors.nome }}
   </div>

   <div *ngIf="formErrors.cpf && submitted" class="alert alert-danger">
       {{ formErrors.cpf }}
   </div>

   <div *ngIf="formErrors.dataNasc && submitted" class="alert alert-danger">
       {{ formErrors.dataNasc }}
   </div>

   <div *ngIf="formErrors.salario && submitted" class="alert alert-danger">
       {{ formErrors.salario }}
   </div>


5. Para desabilitar o botão Enviar caso o formulário esteja inválido:

   <!-- [disabled]="!usuarioGroup.valid" -->


6. Ver código de AppComponent
